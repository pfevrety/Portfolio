Configuration
-----------
You need to install dependencies
```bash
$ yarn install
```
You can run project using
```bash
$ yarn dev
```
You can run project for production using:
```bash
$ yarn build
$ yarn start
```
Generate static project
```bash
$ yarn generate
```
